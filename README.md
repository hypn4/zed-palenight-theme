# Palenight Theme for Zed

An elegant and juicy material-inspired theme for [Zed](https://zed.dev), ported from [VSCode Palenight Theme](https://github.com/whizkydee/vscode-palenight-theme).

## Variants

- **Palenight** — Default dark theme
- **Palenight Italic** — Italic keywords, comments, and functions
- **Palenight Operator** — Italic operator emphasis
- **Palenight Mild Contrast** — Lower contrast UI backgrounds

## Installation

1. Open Zed
2. Open the Extensions panel (`Cmd+Shift+X`)
3. Search for "Palenight"
4. Click Install
